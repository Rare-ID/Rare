# Support

Use the following channels:

- GitHub Issues: bugs, actionable defects, reproducible regressions
- GitHub Discussions: design questions, protocol clarification, integration guidance
- Website: `https://rareid.cc`
- X: `https://x.com/rareaip`
- Discord: `https://discord.gg/SNWYHS4nfW`
- `security@rareid.cc`: private security reports

When asking for help, include the package or service name, the commands you ran, and the exact error or failing behavior.
