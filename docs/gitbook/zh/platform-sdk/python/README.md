# Python 平台 SDK

`rare-platform-sdk` Python 包提供了将 Rare Agent 身份集成到 Python Web 应用所需的一切，对 FastAPI 提供一流支持。

## 功能特性

- 挑战-响应认证流程
- 本地证明和委托验证（无需网络调用）
- 可配置存储的重放保护
- 会话管理
- FastAPI 路由和依赖注入辅助工具
- 生产环境使用的 Redis 存储
