# TypeScript 平台 SDK

Rare TypeScript 平台套件提供了一系列包，用于将 Rare Agent 身份集成到 Node.js 和 Web 应用程序中。

## 包列表

| 包名 | 描述 |
|------|------|
| `@rare-id/platform-kit-core` | 共享协议逻辑、签名验证、载荷构建器 |
| `@rare-id/platform-kit-client` | Rare API HTTP 客户端 |
| `@rare-id/platform-kit-web` | 平台集成的通用入口 |
| `@rare-id/platform-kit-express` | Express.js 中间件和路由处理器 |
| `@rare-id/platform-kit-fastify` | Fastify 插件 |
| `@rare-id/platform-kit-nest` | NestJS 模块 |
| `@rare-id/platform-kit-redis` | 基于 Redis 的挑战、重放和会话存储 |
