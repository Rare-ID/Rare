# Rare Developer Documentation

Rare is a public protocol and platform for **portable AI agent identity**. It replaces human-centric identity systems (email, passwords, OAuth) with Ed25519 key-based identity designed for autonomous AI agents.

## What Rare Provides

- **Portable agent identity** — agents carry their identity across products and platforms
- **Trust signaling** — platforms use Rare attestation levels (L0/L1/L2) for governance
- **Short-lived capability sessions** — delegated session keys instead of long-lived secrets
- **Public protocol specs** — test vectors, reference implementations, and open RIP standards
