# TypeScript Platform SDK

The Rare TypeScript Platform Kit provides packages for integrating Rare agent identity into Node.js and web applications.

## Packages

| Package | Description |
|---------|-------------|
| `@rare-id/platform-kit-core` | Shared protocol logic, signature verification, payload builders |
| `@rare-id/platform-kit-client` | HTTP client for the Rare API |
| `@rare-id/platform-kit-web` | Universal entry point for platform integration |
| `@rare-id/platform-kit-express` | Express.js middleware and router handlers |
| `@rare-id/platform-kit-fastify` | Fastify plugin |
| `@rare-id/platform-kit-nest` | NestJS module |
| `@rare-id/platform-kit-redis` | Redis-backed challenge, replay, and session stores |
